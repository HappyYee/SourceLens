# Roadmap

## P0: Must Finish First

- Keep repository private and safe from secrets/local data.
- Fix X login status check misreporting `expired`.
- Improve X profile-busy user-facing errors.
- Keep YouTube, Bilibili, and X refreshes stable.
- Preserve custom titles and local user edits during refreshes.

## P1: Main Features

- Add an X Debug Panel for observability.
- Improve X quote-card extraction.
- Add clearer refresh/backfill diagnostics per source.
- Improve Room source management ergonomics.
- Add focused tests around browser-profile error handling.

## P2: Optimization and Experience

- Better empty/loading/error states across Room and settings screens.
- More resilient thumbnail loading for platform media.
- Optional source packages or templates.
- Export/import improvements.
- Better local deployment documentation with Cloudflare Tunnel or equivalent.

## P3: Future Ideas

- Multi-device read-only access with password protection.
- More platform connectors.
- Local model-assisted summarization or title generation.
- Read-later workflows.
- Personal analytics focused on "finished reading" rather than engagement.
